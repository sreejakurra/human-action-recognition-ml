# Human Activity Recognition using Machine Learning

## Objective
To classify human activities based on sensor data collected from wearable devices.

## Description
This project focuses on Human Activity Recognition (HAR), where machine learning techniques are used to identify activities from sensor signals such as accelerometer and ECG data.

The dataset contains readings from multiple sensors placed on the body (e.g., chest, ankle), and the goal is to classify different physical activities based on these signals.

## Approach
- Performed data preprocessing and cleaning
- Handled multiple sensor features (X, Y, Z axes)
- Trained a machine learning model (Random Forest Classifier)
- Evaluated model performance using accuracy and confusion matrix

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results
The model was able to classify human activities with good accuracy, demonstrating the effectiveness of machine learning in sensor-based activity recognition.

## Conclusion
This project helped in understanding real-world applications of machine learning in healthcare and activity monitoring systems.
